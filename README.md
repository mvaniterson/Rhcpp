# Rhcpp
A fast R implementation using Rcpp based on the original matlab HCP
 method for Normalizing RNA-Sequencing Data by Modeling Hidden Covariates with
 Prior Knowledge by Mostafavi et al. PLOSone (2013).
# Install

From unix-alike use either:

```r
library(devtools)
install_github("mvaniterson/Rhcpp")
```
or using the biocLite function downloaded from BioConductor.

```r
source("http://bioconductor.org/biocLite.R")
biocLite("mvaniterson/Rhcpp")
```
# Usage

see ?hcp or unitTest directory. The package contains example data.
